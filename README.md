Este repositório contém a proposta de API instanciado pela atividade da primeira semana do curso de NodeJs da etapa de treinamento da InJunior.

Aqui você encontrará uma API simples, que gerencia usuários e posts em uma escala de 1:N (ou seja, vários posts para um usuário), e possui alguns caminhos criados em união aos métodos HTTP.

Nesse caminho você encontrará o caminho para o arquivo postman de uso dos métodos !!Antes de tentar usa-los dê o comando "npm run start:dev" no terminal:
[Clique aqui para ver as rotas de post](AulasBackPostman.json)
